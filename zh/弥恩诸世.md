在此地住民的口中，**弥恩**即为“天地”之意。由此，**弥恩文明**便成为他们用以自称的称谓。

**弥恩文明**自诞生之初便与恶意相伴——诡异的{{古孽|/弥恩诸世/术语汇编/古孽}}盘踞在疆域之外，狂暴的{{龙族|/弥恩诸世/术语汇编/龙族}}潜伏于深山林涧，古怪的{{智械|/弥恩诸世/术语汇编/智械}}镇守着古代废墟……在这被统称为{{灾厄|/弥恩诸世/术语汇编/灾厄}}的三重威胁环伺之下，弥恩文明顽强地生存、成长。历经漫长的 **221 纪年**，其疆域从**昭旦平原**向东扩张至**无垠沧海**，向南抵达**广袤雨林**，向北延伸至**连绵沙碛**，更向西将旗帜插到了以厄南峰为核心的**天山山脉**脚下。

在这片面积超过六千万平方公里的辽阔土地上，分布着多个以“{{中央建构|/弥恩诸世/术语汇编/中央建构}}”为核心的大区。辽阔的疆域、驳杂的人口与漫长的地方历史，使各区在政治、经济、文化与信仰上呈现出截然不同的面貌。而曾经统御诸区、率领众生抗击灾厄的，是一个永恒而伟大的帝国——以文明之名冠称的**弥恩帝国**。

然而，**七纪年**前，帝国倾覆，辉煌的统一文明化为齑粉，皇城覆灭，天断其方。皇室血脉连同帝制法统一道，在阴谋、战争与遗忘中断绝。破碎山河间，军阀、城邦、国度如野草般滋生，在昔日帝国的遗骸之上挣扎求生。

**弥恩**那漫长的**童年时期**已过，而遭无可抗阻的**历史浪潮**割裂——社会与自然的矛盾如此尖锐，迫使世人再不能安于现状，只得被卷入无休止的斗争之中。在这**山河颠倒**、**社会聚变**的时代中，将涌现出多少英雄豪杰，又潜伏着多少虎豹熊罴。历史的漫漫长夜，何日才能停止它的呜咽？

一切的故事，都要自这混沌世间拉开序幕。

# 世界基调

**弥恩**并非传统意义上的英雄奇幻世界。其核心基调为**近代魔幻现实主义**：魔法、神术与奥法并未消失，只是失去了其古代神秘学的外衣，而在生产力与生产关系的近代发展中成为可被研究和管理的普遍公共理性。在弥恩世界，个人武力并不能轻易凌驾于历史之上。无论是强大的施法者、战士，还是身居高位的统治者，都必须面对经济与自然共同构成的庞然矛盾。

在这样一个世界中，冒险并不只是“击败邪恶”或“获取宝藏”，而将是玩家角色与历史进程发生联系的过程。玩家角色仍能成为英雄，但这种英雄性并不来自超脱时代的宿命，而来自他们在具体矛盾中的行动、选择与承担。

因此，弥恩战役是更为**现实主义**的，玩家行动将在具体的历史进程中，与确实存在的**社会**或**自然**矛盾交织，并综合他们在曲折道路中的主观能动性，用以展现个人行动与社会进程的辩证关系。奇迹、冒险与史诗感依然存在，但并非孤立，而应当扎根于真实可感的社会环境之中。

# 天文历法

**弥恩通行历法**依据芙莱（此方太阳）、伊莱（此方月亮）、维洛斯星带与宇域神宿的运行规律制定。弥恩诸地以三十六小时为一日，以伊莱的一次完整月相周期为一月；每经过六十四月，弥恩完成一次环绕芙莱的公转，此一轮回即称为一**纪年**。

每纪年之始，以**启明之星**、**弥恩**与**芙莱**三星共线为基准，此日被定为**纪元日**。由于公转周期并非恰好等同于六十四月，弥恩诸地亦有**定纪**与**置闰**之制，用以校准历法。

在纪年与月之间，弥恩历法另设**节历**。每八个月为一**时节**，一纪年共分八节，依次为**启蛰**、**芳华**、**清霖**、**炎焚**、**素秋**、**霜降**、**玄英**与**归藏**。

一套标准的弥恩历法日期，其书写格式为：`XXX纪年 · XX时节 · X月X日`。

# 魔网体系

弥恩与现实世界最根本的差异，在于**魔网**的客观存在。魔网并非单纯的神秘象征，也不是少数施法者独占的个人天赋，而是一种能够被感知、研究、改造和利用的客观规律。

在古代，魔法被视为一种偶发的个人天赋，而进入近代以后，随着自然科学与社会结构的发展，魔网——广义上来说是一切的自然规律——都得到了祛魅，被逐渐纳入现代知识体系之中。

与大多数奇幻世界不同，弥恩的魔网并不稳定。奥术与神术可能失效，甚至时而引发灾难性的后果。魔网既是弥恩文明得以发展壮大的基础，也是其始终无法摆脱的自然压力。

# 位面模型

**弥恩**所在的世界，是一个纯粹而封闭的单一世界。所有已知故事均发生于唯一的**物质位面**，亦即通常所称的**主位面**之内。不论地底深处的幽暗地域、苍穹之上的神域、古代遗迹中的封闭空间，乃至由**魔法**短暂构筑或稳定维持的各类额外空间，都归属于主位面内。换言之，凡可被观测、抵达、占据的空间，皆被视为**物质位面**的一部分。

与此相对，所谓**非物质位面**在弥恩始终处于未定状态。其概念主要源于对**异界存在**的研究：无论是弱小的**元素生物**，还是强大的**异界宗主**，它们都能经由**魔网**被召唤至物质世界，并在其中产生真实可见的影响。然而，这些存在**不能**准确描述自身来源之地。任何异界存在一旦试图叙述非物质位面的形态、结构或方位，便会陷入不可逆的思维错乱；而物质位面的生灵也无法通过传送、观测或其他法术手段抵达或验证这些领域。

因此，弥恩的位面理论并不承认“可被访问的多重世界”。它更倾向于认为一切实体存在皆依附于**物质位面**，而所谓非物质位面只是与异界力量相关的概念性解释。它们或许能够被推理和调用（并在规则层面产生效应），却不构成可供直接探索的空间。

在实际研究中，弥恩学界仍会为了解释某些**次元魔法**、召唤法术、遣返效应或异界神秘现象，建立多种“位面”模型。这些模型并非来自实地观测，而是根据魔法效果反推而成的理论框架。

# 经济生产

**弥恩**是一片广袤而分裂的大陆，各地**政治生态**迥异，**经济发展水平**亦呈现出显著差别。这些区域与阶层之间的**经济差别**，往往是驱动政治行动的首要因素。因此，**玩家角色**在弥恩诸地的冒险中，不仅要面对对生命的直接威胁，也会不断受到**经济环境**的影响。一条商路是否畅通，一座矿山由谁控制，一地粮价如何变动，都可能改变冒险的动机与结果。玩家角色的行动不应只被理解为道德判断或宿命召唤，也应与其在现实生活中的**得失**相关：报酬、债务与组织资源，以及他们试图保护或改变的社会关系，都会成为推动冒险的重要力量。

为了更好地契合弥恩的基调，并赋予玩家更具象的行动动机，弥恩世界相较于《玩家手册》（PHB）对**财产规则**作出了一系列调整，包括**货币体系**的变更、**标准物价**的修订，以及对不同地区**经济差异**的补充说明，以使生产、交换与剥削关系能够更具体地进入叙事与游戏过程之中。

# 意识形态

  在**弥恩**世界，意识形态是**社会存在**——特定历史阶段下的物质生产方式、阶级斗争格局、权力结构以及人类与自然的实践关系——在**观念领域**，也即**社会意识**的**系统性、结构性反映**。
  
 为刻画这种驱动集体与个人行动的深层逻辑，**弥恩**世设以具体矛盾为依据，对个人、组织、政权与社会运动的思想和行动方式进行分类，并在不同轴向上做出两轴区分，以深入本质的形式反应政治实体、社会运动乃至重要人物的**根本哲学立场**。这将作为地下城主分析势力行为、推演矛盾发展与规划历史进展的核心工具，也是玩家理解所处世界、做出符合角色世界观选择的重要参考。

  意识形态会在具体的历史条件下生成，与阵营不同，意识形态的分类一定是准确的，进步与保守黑白分明，不存在中立。对于特定时空下的实体，其意识形态必定具有相对确定的状态，反映其在此时此刻面对具体矛盾时所采取的立场。并且，意识形态会随着**具体社会实践的进行**而持续演化——也即，意识形态的立场与具体的社会实践是不可分割的，考虑单一意识形态的进步或保守，必须结合当前的社会实景。

# 信仰体系
**信仰体系**是自意识形态之外支配着弥恩社会实体世界观的一大因素，与意识形态源于对社会与自然矛盾的理性认知和立场选择不同，信仰体系更多关乎于文化传承、超验体验、集体认同以及对“不可知”的解释。**神秘力量**的客观存在，确保了信仰体系在弥恩不会如同其在现实世界那样轻易消退，其有时甚至会作为意识形态的**主体或基石**，对实体的行动或观念造成显著影响。然而，工业化、民族国家认同与激烈的阶级分化，都在冲击和稀释着传统的信仰体系。

正如同宗教或文化信仰在现实世界中的地位一致，弥恩的信仰体系一方面是一种深层的**文化遗产**和**集体无意识**，塑造了各地区的习俗、道德观和身份认同，另一方面，又作为一种组织化的**社会力量**与**利益集团**，确实地参与社会实践并受社会实践的反向支配，它的地位是复杂而多变的，不像意识形态那样清晰明了，用于描述个体也缺乏概括意义。

总的来说，在弥恩世界，信仰已从**支配性**的绝对真理，转向了**参与性**的社会文化因素之一。它时而被意识形态吸收或改造，时而与之产生紧张对抗，或是淡化出社会实践的场域。
