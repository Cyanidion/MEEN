# 行道大劫
***221纪年玄英时节1月30日 - 2月3日***

{{Quote|"皇室亡灭，帝国倾覆，豺狼喧嚣，山河破碎。
七纪以来，弥恩之地，诸侯割据，狼烟四起。奸佞谗毁，贪权窃柄。邪霸猾孽，巧取豪夺。严刑峻法，穷兵黩武。强征壮丁，妄动国政。奴隶苍生，牲畜天下。致使耕地荒废，饿殍遍地。百姓凋悴，颠沛流离。十室九空，千里无人。白骨埋道，坟茔聚山。
乱世之中，唯我天府。尊王讨奸，匡扶社稷。上下一心，秣马厉兵。马踏火隅，枪击汐暝。一统东地，守望黎民。休养生息，百姓安康。终有路不拾遗，歌舞升平。
然天下未定，海内未平。贵胄既寻偏安，商贾但求谋利。把持朝野，藐视上意。花天酒地，声色犬马。效亡灭之前鉴，行暗弱之故事。
尔曹出身大家，富可敌国。虽学富五车，身怀绝技，然行尽堕落，毫无羞耻。外不闻天下危亡，内不顾苍生倒悬。潇洒畅快，恣意人生，花前月下，酒池肉林。可叹不知快活，仍有几何？"|祝酒词}}

<p>221纪年玄英时节1月30日，执权东赐60余年的{{守望者|/弥恩诸世/术语汇编/守望者}}<a href="/弥恩诸世/人物汇总/莫兰德·福尔斯">莫兰德·福尔斯</a>被{{咨政院|/弥恩诸世/术语汇编/咨政院}}与{{总参谋部|/弥恩诸世/术语汇编/总参谋部}}宣告过世。</p>
<p>对许多东赐人来说，这是旧时代彻底结束的标志，也是朝新时代前进的契机，在经历了几十年的平静后，人心似乎都在思变。作为影子政府控制东赐政治经济将近40年的{{五大家|/弥恩诸世/术语汇编/五大家}}也都蠢蠢欲动，纷纷以不同方式策划夺取东赐的最高权力。</p>
<p>2月3日，在各方的明争暗斗下，多方的冲突在行道城集中爆发。<a href="/弥恩诸世/人物汇总/斯卡普·哈茨基">斯卡普·哈茨基</a>率领{{隅南帝国军|/弥恩诸世/术语汇编/隅南帝国军}}攻入行道城，于首都的街巷上与{{东赐帝国军|/弥恩诸世/术语汇编/东赐帝国军}}展开了激烈的交战。于此同时，圣庭于宫城中背弃了<a href="/弥恩诸世/人物汇总/里维斯·维西尼亚">里维斯·维西尼亚</a>，使用{{奥切安之匣|/弥恩诸世/增改规则/魔法、高科物品/奇物/奥切安之匣}}开始了魔网破坏仪式。</p>
<p>在战火、灾害、混乱、背叛、屠杀与浴血中，以<a href="/弥恩诸世/人物汇总/让娜·洛托">让娜·洛托</a>为代表的一方，突袭进入宫城，击败圣庭领袖<a href="/弥恩诸世/人物汇总/白岛·爱斯菲尔德">白岛·爱斯菲尔德</a>，并率领东赐帝国军残部抵御隅南帝国军进攻，成功夺取最终胜利。</p>
<p>但本次事件对行道城造成了极大的破坏，造成的直接经济损失高达千万守望币。有数百年历史的宫城遭到严重破坏，居民区与工业区大面积化为废墟，东赐帝国军兵力严重损失，大量居民、企业宣告破产，死伤者更是高达数十万，亲历者中不少人称呼当时的场景为”人间炼狱“。此外，由于魔网稳定度的剧烈波动，大量居民、作物与牲畜等罹患上疾病，对生产生活造成了极大影响。</p>
<p>该事件最终被洛托政府命名为“行道大劫”。</p>

# 事件经过

<blockquote class="custom-blockquote-default">
<p>221纪年玄英时节1月29日，莫兰德过世。在与<a href="/弥恩诸世/人物汇总/多伊·阿庇奥">多伊·阿庇奥</a>所代表的圣庭进行秘密交易后，里维斯·维西尼亚依靠家族在军队中的势力，开始了针对其他三大家的政治迫害，而多伊也策划了多起恐怖袭击。</p>
<p>29日起，城内的犯罪案件激增，抢劫、袭警、绑架、谋杀等案件激增。其中，帝国军{{参谋总长|/弥恩诸世/术语汇编/参谋总长}}<a href="/弥恩诸世/人物汇总/伊莱克尼·洛托">伊莱克尼·洛托</a>及若干咨政院议员、帝国军高层遭到刺杀，许多政要也遭遇威胁与排挤。<a href="/弥恩诸世/人物汇总/维克托·弗里曼">维克托·弗里曼</a>等维西尼亚心腹开始控制咨政院与总参谋部。</p>
<p>30日，{{顾问总理|/弥恩诸世/术语汇编/顾问总理}}<a href="/弥恩诸世/人物汇总/安德烈娅·加拉诺德尔">安德烈娅·加拉诺德尔</a>遭到里维斯的非法罢免，政权在阴谋中彻底落入里维斯之手。</p>
<p>2月1日，里维斯宣布当选新任守望者，让娜等人遭到通缉。“<a href="/弥恩诸世/组织/新东赐">新东赐</a>”地下残部遭到剿灭。</p>
<p>2日，里维斯举行{{授剑仪式|/弥恩诸世/术语汇编/授剑仪式}}，同时按照约定，魔网圣神信仰被确定为国教。同日，安德烈娅与瑞斯克发布了著名的“{{觉醒演讲|/弥恩诸世/术语汇编/觉醒演讲}}”。</p>
<p>3日，斯卡普率领隅南帝国军突袭攻城，并从边界线进军。城市化为战火纷飞的战场。宫城中，新任守望者里维斯遭到圣庭领袖白岛背叛，开启魔网破坏仪式。让娜·洛托、<a href="/弥恩诸世/人物汇总/瑞斯克·戴斯">瑞斯克·戴斯</a>、<a href="/弥恩诸世/人物汇总/安以赛亚·加拉诺德尔">以赛亚·加拉诺德尔</a>及<a href="/弥恩诸世/人物汇总/卡特·索恩">卡特·索恩</a>、<a href="/弥恩诸世/人物汇总/卡拉克·帕德尔">卡拉克·帕德尔</a>、<a href="/弥恩诸世/人物汇总/柯尼·莱斯">柯尼·“屠龙者”·莱斯</a>突袭进入宫城，击杀白岛，随后率领东赐帝国军围绕宫城展开了殊死抵抗。</p>
<p>4日，东赐方面增援抵达，隅南方面撤军。</p>
