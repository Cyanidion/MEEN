# 帝国时期
***0纪年 —— 214纪年芳华时节5月***

帝国时期是对**古代**与**近代**的合称，即自历法元纪，至{{天堂坠落|/弥恩诸世/历史沿革/历史事件/天堂坠落}}的这一段时期。

# 详述
帝国时期以弥恩帝国的建立、延续与衰落为主线。自[科朗森·埃斯托内科](/弥恩诸世/人物汇总/科朗森·埃斯托内科)建立帝国并制定历法元纪后，弥恩文明从原本的部落联盟形态进入以{{弥恩皇室|/弥恩诸世/术语汇编/弥恩皇室}}为法统核心的帝国时代。

帝国时期内部包含{{开拓年代|/弥恩诸世/历史沿革/历史时期/开拓年代}}、{{恒辉时代|/弥恩诸世/历史沿革/历史时期/恒辉时代}}与{{灾后时代|/弥恩诸世/历史沿革/历史时期/灾后时代}}。

其中，开拓年代对应帝国建立后文明快速扩张的早期阶段；恒辉时代对应{{闪星湮灭|/弥恩诸世/历史沿革/历史事件/闪星湮灭}}后帝国秩序长期稳定维持的阶段；灾后时代则对应{{第一次天灾|/弥恩诸世/历史沿革/历史事件/第一次天灾}}后帝国受到重创、旧有秩序迅速动摇的阶段。

在帝国时期内，帝国法典、皇室法统、军务体系与地方议会共同构成弥恩主要政治秩序。尽管帝国在不同时段、不同地区的实际控制能力并不一致，但弥恩皇室仍长期被视为全弥恩最主要的法理统治核心。

第214纪年，天堂坠落发生，新诺阿坠入深渊，帝国统治中枢遭到毁灭性破坏，帝国时期由此结束。
