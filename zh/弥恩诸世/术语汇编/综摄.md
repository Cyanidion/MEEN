# 综摄

**综摄**是[共和政府](/弥恩诸世/政权组织/弥恩联区共和国)的政府首脑，拥有极大权力，是国家权力核心。当前，该职由[让娜·洛托](/弥恩诸世/人物汇总/让娜·洛托)担任。

# 详述

221纪年玄英时节2月的{{洛托新政|/弥恩诸世/术语汇编/洛托新政}}中，{{咨政院|/弥恩诸世/术语汇编/咨政院}}与{{顾问总理|/弥恩诸世/术语汇编/顾问总理}}被废除，政府首脑改为由{{大议会|/弥恩诸世/术语汇编/大议会}}多数党领袖担任的综摄。

{{《弥恩共和国宪法》|/弥恩诸世/术语汇编/《弥恩共和国宪法》}}规定：
1. 综摄为弥恩共和国政府首脑，
2. 综摄由大议会最大党或执政联盟领袖担任；
3. 综摄有权任免政务团各部长；
4. 综摄负责主持政务团会议，统筹政务工作；
5. 综摄与政务团各部长集体管理国内宏观与微观层面行政。
6. 综摄职权主要包括：
- 主持并参加政务团会议；
- 副署政务团法案与政令；
- 向大议会汇报政务团工作并接受质询；
- 其他政务团成员拥有的职权。

让娜在实施新政后宣布自己以“临时代行”的名义就任第一任综摄。
