# 三大家

**三大家**，或“**新三家**”通常指，在221纪年玄英时节后主导[东赐政府](/弥恩诸世/政权组织/弥恩帝国（东赐政府）)政局的三人所在家族。

# 详述

三大家包括：
- {{综摄|/弥恩诸世/术语汇编/综摄}}[让娜·洛托](/弥恩诸世/人物汇总/让娜·洛托)
- {{统领|/弥恩诸世/术语汇编/统领}}[瑞斯克·戴斯](/弥恩诸世/人物汇总/瑞斯克·戴斯)
- {{民主共和党|/弥恩诸世/术语汇编/民主共和党}}全国委员会主席[以赛亚·加拉诺德尔](/弥恩诸世/人物汇总/以赛亚·加拉诺德尔)所在的家族。

虽然民间将其称为三大家，但不同于过去的{{五大家|/弥恩诸世/术语汇编/五大家}}，三人既没有以家族为单位，进行对政权或经济、产业的强硬控制，其内部也呈现以让娜为绝对核心的不平等地位。
