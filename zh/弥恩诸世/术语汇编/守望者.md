# 守望者

**守望者**是[东赐政府](/弥恩诸世/术语汇编/东赐政府)莫兰德时期的国家元首及政府首脑，行使军事独裁的绝对权力。一般而言，即指[莫兰德·福尔斯](/弥恩诸世/人物汇总/莫兰德·福尔斯)。

# 详述

214纪年·芳华时节{{天堂坠落|/弥恩诸世/历史沿革/历史事件/天堂坠落}}发生后不久，帝国勋爵[莫兰德·福尔斯](/弥恩诸世/人物汇总/莫兰德·福尔斯)便宣布废除帝国{{督理|/弥恩诸世/术语汇编/督理}}及{{监察|/弥恩诸世/术语汇编/监察}}，实行{{守望体制|/弥恩诸世/术语汇编/守望体制}}，并担任集国家元首与政府首脑于一身的守望者，实行个人的军事独裁统治。
216纪年·霜降时节{{皇冠事变|/弥恩诸世/历史沿革/历史事件/皇冠事变}}后，莫兰德被迫放弃个人的军事独裁，将权力交与{{五大家|/弥恩诸世/术语汇编/五大家}}控制下的{{咨政院|/弥恩诸世/术语汇编/咨政院}}与{{参谋部|/弥恩诸世/术语汇编/参谋部}}，成为无实权的花瓶国家元首。
221纪年·玄英时节{{行道大劫|/弥恩诸世/历史沿革/历史事件/行道大劫}}中，[里维斯·维西尼亚](/弥恩诸世/人物汇总/里维斯·维西尼亚)与[多伊·阿庇奥](/弥恩诸世/人物汇总/多伊·阿庇奥)合作，通过政治暴力与阴谋活动，将其余三大家排挤出权力核心，攫取了{{顾问总理|/弥恩诸世/术语汇编/顾问总理}}与{{总参谋长|/弥恩诸世/术语汇编/总参谋长}}的权力，重新恢复了守望者的个人军事独裁权力。但其随后在宫城被圣庭领袖[白岛·爱斯菲尔德](/弥恩诸世/人物汇总/白岛·爱斯菲尔德)杀害。
{{洛托新政|/弥恩诸世/历史沿革/历史事件/洛托新政}}中，守望者被废除，改设无实权的{{统领|/弥恩诸世/术语汇编/统领}}作为国家元首。
