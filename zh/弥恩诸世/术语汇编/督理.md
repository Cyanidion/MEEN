# 督理

**督理**是{{帝国时期|/弥恩诸世/术语汇编/帝国时期}}在各大区设置的最高名义职务之一，其职责是统领所在地区的{{帝国军|/弥恩诸世/术语汇编/帝国军}}，负责地区军务、防卫部署、军队调度以及与地方行政机构之间的军事协调。

# 详述

从法理上看，督理不掌握地方行政权，不是地方行政首脑，不直接负责税赋、司法、民政、商贸管理或官员任免。地方行政事务仍由各大区自身的行政机构承担，督理则负责帝国军在地区内的组织、调度与防卫事务。但在实际政治中，督理的影响力取决于地区军务的重要性和地方行政机构的强弱。边疆地区、战乱地区或灾厄频发地区往往更依赖帝国军维持秩序，督理也因此更容易取得超出法理职权的现实权力。督理通常由本地区已经拥有军政资源的人物担任，其名义军权常常会与地方原有实权结合，使督理在不同大区具有不同程度的实际影响。
