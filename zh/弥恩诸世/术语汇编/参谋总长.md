# 参谋总长

**参谋总长**是[东赐政府](/弥恩诸世/政权组织/弥恩帝国（东赐政府）)五大家时期的军事首领。

# 详述
在{{天堂坠落|/弥恩诸世/历史沿革/历史事件/天堂坠落}}发生后的214纪年·芳华时节·5月16日，帝国勋爵[莫兰德·福尔斯](/弥恩诸世/人物汇总/莫兰德·福尔斯)宣布于东赐建立{{守望体制|/弥恩诸世/术语汇编/守望体制}}。{{帝国军|/弥恩诸世/术语汇编/帝国军}}被改组为具有浓厚{{军阀主义|/弥恩诸世/意识形态/军阀主义}}色彩的部队，并建立{{总参谋部|/弥恩诸世/术语汇编/总参谋部}}及各级参谋部，负责协助{{守望者|/弥恩诸世/术语汇编/守望者}}管理新生的{{东赐帝国军|/弥恩诸世/术语汇编/东赐帝国军}}，协助进行军队训练、调度、征召等日常事务。
这一时期，参谋总长作为总参谋部与武官领袖，由守望者直接任命。在守望者的授权下，参谋总长可以直接参与东赐政府各项军事事务的运行，但其自主裁量权与自由权力十分有限，大部分时候都被视为守望者的传令官。
216纪年·霜降时节{{皇冠事变|/弥恩诸世/历史沿革/历史事件/皇冠事变}}后，守望者[莫兰德·福尔斯](/弥恩诸世/人物汇总/莫兰德·福尔斯)被迫放弃独裁并遭到软禁，东赐的军事大权转移到以时任参谋总长[普莱提克·维西尼亚](/弥恩诸世/人物汇总/普莱提克·维西尼亚)为首的总参谋部手中。
参谋总长由此成为东赐政府军事首领，拥有受咨政院节制的最高军事权。其产生方式改由咨政院内部选举产生，四年一届，不得再任，于每年2月1日进行选举。实际运行中，由于{{五大家|/弥恩诸世/术语汇编/五大家}}影子政府对总参谋部的控制，历届参谋总长大多亦只是五大家的代言人。
221纪年·玄英时节{{洛托新政|/弥恩诸世/术语汇编/洛托新政}}后，参谋总长与帝国军改由{{综摄|/弥恩诸世/术语汇编/综摄}}直接管辖，由综摄直接任命。
