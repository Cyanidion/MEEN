# 统领

**统领**是[共和政府](/弥恩诸世/政权组织/弥恩联区共和国)的国家元首，无实权，主要为礼仪性和象征性角色。当前，该职由[瑞斯克·戴斯](/弥恩诸世/人物汇总/瑞斯克·戴斯)担任。

# 详述

221纪年玄英时节3月的{{洛托新政|/弥恩诸世/术语汇编/洛托新政}}中，{{守望者|/弥恩诸世/术语汇编/守望者}}被废除，国家元首改为由普选产生的统领。

{{《弥恩联区共和国宪法》|/弥恩诸世/术语汇编/《弥恩联区共和国宪法》}}规定：
1. 统领为弥恩联区共和国元首，是大区凝聚象征；
2. 统领由普选获得绝对多数票者担任，如无人获得绝对多数，获票最多的两名候选人进行权变选举，由大议会进行第二次选举，获得多数票者当选；
2. 统领任期五年，可连任一届；
3. 统领职权主要包括：
- 代表联区政府，参与联区活动；
- 宣布缔约、同盟、断交、战争、媾和；
- 协助综摄开展政府工作。

[让娜·洛托](/弥恩诸世/人物汇总/让娜·洛托)实施新政后的第一次全国统领选举中，[民主共和党](/弥恩诸世/术语汇编/民主共和党)候选人[瑞斯克·戴斯](/弥恩诸世/人物汇总/瑞斯克·戴斯)击败[弥恩党](/弥恩诸世/术语汇编/弥恩党)候选人[提格雷·纽斯卡尔](/弥恩诸世/人物汇总/提格雷·纽斯卡尔)，当选第一任统领。
