# 帝国军

帝国军是对弥恩帝国境内所有军队的统称，其肩负着守护弥恩文明的重任。

# 详述

帝国军是{{弥恩皇室|/弥恩诸世/术语汇编/弥恩皇室}}最重要的直属组织，也是皇室权威的主要现实基础。依据{{《帝国法典》|/弥恩诸世/术语汇编/《帝国法典》}}，弥恩帝国皇帝对帝国军拥有最高指挥权、决定权与任免权。由于皇室在法理上主要掌握军权，因此帝国军并不只是一个普通军事组织，而是皇室得以维持其法统地位、灾厄防卫职责和文明守护叙事的核心支柱。

帝国军的基本使命是保护全弥恩文明的安定，尤其是抵御灾厄侵蚀和处理高危地区的安全威胁。在弥恩的政治结构中，皇室并不直接管理所有地方行政事务，但帝国军却可以凭借{{灾厄|/弥恩诸世/术语汇编/灾厄}}防卫的名义部署在边境、危险地区和部分战略节点。帝国军所驻地的普通民众有义务向军队提供物资支持，这一制度被称为{{收纳援助|/弥恩诸世/术语汇编/收纳援助}}。

帝国军内部包含多个层级和分支。{{皇室卫队|/弥恩诸世/术语汇编/皇室卫队}}和{{皇家近卫法师团|/弥恩诸世/术语汇编/皇家近卫法师团}}由最精锐的士兵与法师组成，作为皇帝直接指挥的近卫力量服役，承担保护皇帝、守卫皇室核心区域和执行高风险军务的职责。{{巡守卫队|/弥恩诸世/术语汇编/巡守卫队}}则主要在各大区协助治安，尤其用于应对破坏性较高、地方机构难以及时处理的内部威胁。正规编制的帝国军则更多部署在边境和危险地区，负责长期防御灾厄、维持军事据点和保障文明边界。

在皇室强盛时期，加入帝国军是普通人改变命运的重要途径之一。无论出身于哪个地区、哪个行业，军队人员通常都受到广泛尊重，因为他们被视为“弥恩之盾”，以自身牺牲保卫文明延续。帝国军也因此不仅具有军事意义，还具有强烈的社会象征意义。它代表着皇室仍然能够站在灾厄之前，也代表着普通人可以通过军功、服役和牺牲进入帝国公共秩序的核心。

天灾以后，帝国军遭受了极为沉重的打击。大量精锐部队、巡守力量和边境军备在灾难中损失，皇室卫队与皇家近卫法师团规模缩减九成，曾经遍布大陆的巡守体系也只能勉强维持中部地区的治安稳定。与此同时，皇室产业大量外售，帝国财产不断流失，使帝国军的财政供养越来越难以维持。帝国军仍然保有皇室直属武装的法理地位，却已经不再具备过去那种覆盖全弥恩、压制大型危机和稳定边境局势的完整能力。

[蔓娜莱莎女皇](/弥恩诸世/人物汇总/蔓娜莱莎·埃斯托内科)执政期间，帝国军被重新推入高度动员状态。女皇以复仇为核心目标，发起大规模“远征”行动，在各地收拢政权、征用超额援助，并试图以集权方式重建皇室军事能力。这一过程短时间内强化了帝国军的动员规模，却也严重透支了民众的支持和军队自身的恢复能力。[罗西塔亲王](/弥恩诸世/人物汇总/罗西塔·埃斯托内科)摄政后，帝国军转入收缩与维持状态，边疆军备依靠变卖皇室产业勉强维系，部分地区的{{监察|/弥恩诸世/术语汇编/监察}}与{{督理|/弥恩诸世/术语汇编/督理}}权限也被下放给地方议会。

自此以后，帝国军从全弥恩文明的主动防卫核心，逐渐退化为一个仍有法统名义、但现实力量大幅削弱的残存军务体系，并在{{天堂坠落|/弥恩诸世/历史沿革/历史事件/天堂坠落}}后成为地方政权与军阀们的私兵。
