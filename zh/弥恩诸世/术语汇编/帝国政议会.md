# 帝国政议会

帝国政议会是隶属于{{弥恩皇室|/弥恩诸世/术语汇编/弥恩皇室}}的近侧议政机构，主要职责是协助皇帝处理皇室政务与军事政务。

# 详述

帝国政议会本质上是皇帝身边的顾问团、协调机构与军务参议组织，与东赐大区的{{咨政院|/弥恩诸世/术语汇编/咨政院}}具有很高相似性。且由于皇室自身的核心职权集中于军权，因此帝国政议会所处理的事务，也大多围绕{{帝国军|/弥恩诸世/术语汇编/帝国军}}调度、皇室直属组织协调、边境防务、收纳援助分配和帝国财产使用等问题展开。

与帝国政议会偏向议政和协助决策不同，文枢部更接近皇室体系中的事务官僚机构，它不直接制定帝国政策，却负责让皇室已经决定的事项能够被记录、传达、执行和结算。

政议会成员由皇帝直接任免，通常来自各大族、地区显赫人物、帝国军高层或长期服务于皇室的资深官僚。它本身并没有独立于皇帝之外的强制权力，也不能像{{共治议会|/弥恩诸世/术语汇编/共治议会}}那样以制度程序对全帝国事务作出正式决议，但由于其成员能够直接接触皇帝，并参与皇室决策形成过程，因此在实际政治中具有相当重要的影响力。很多时候，政议会并不是通过法定权力来影响帝国，而是通过对皇帝判断、军务优先级和皇室资源分配的影响来发挥作用。

在帝国强盛时期，政议会是皇室与地方精英之间的重要连接点。各地区显赫人物进入政议会，既意味着获得皇帝信任，也意味着其所在地区、家族或军事系统能够在皇室近侧拥有稳定的话语渠道。这种安排一方面使皇室能够吸纳地方信息，避免军务体系脱离现实治理环境；另一方面也使政议会成为各方势力试探皇室态度、争取军务资源和协调利益冲突的场所。

{{第一次天灾|/弥恩诸世/术语汇编/第一次天灾}}以后，政议会的地位发生了明显变化。由于帝国军损失惨重、皇室威信下降，政议会不再拥有过去那种围绕强势皇帝和完整军务体系运转的稳定基础。在[蔓娜莱莎女皇](/弥恩诸世/人物汇总/蔓娜莱莎·埃斯托内科)执政期间，政议会一度被复仇性的集权动员裹挟，其讨论空间和缓冲作用被大幅压缩；[罗西塔亲王](/弥恩诸世/人物汇总/罗西塔·埃斯托内科)摄政以后，随着皇室主动收缩权力、下放部分监察与督理权限，政议会又更多承担起维持残余皇室体系运转、协调产业变卖与边疆军备的事务性角色。

至{{天堂坠落|/弥恩诸世/历史沿革/历史事件/天堂坠落}}前后，帝国政议会已经很难再作为一个有效的皇室中枢机构发挥作用，更多只是象征着旧日皇室秩序最后的组织残余。
