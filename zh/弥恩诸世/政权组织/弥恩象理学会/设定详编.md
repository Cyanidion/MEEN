# **弥恩象理学会（历史与设定）**

### **组织定位与事实主权**

学会并非传统的行政国家，但在地缘政治中被认为拥有事实上的主权地位，这主要源于其在核心能效管理上的技术占有：

*   **魔网稳定锚的控制权**：学会主导了魔网稳定锚的计算模型与稳定算法。若无学会提供的技术支持，跨区域的导魔铁路与大型厂区在面临混乱源魔网冲击时，普遍被认为存在较高的瘫痪风险。
*   **镜玉度量衡与工业规范**：全大陆的镜玉能阶划分与冶炼纯度认证通常由学会发布。不服从该规范的导魔设备可能面临兼容性问题，这使学会实际上掌握了能效度量衡标准。
*   **独立司法与安保权**：根据与部分政权的协议，在象理联络处、联合实验室和边疆调查台站周边特定区域内，学会享有独立的司法管辖权。其专属武装力量象理纠察队被授权对窃取技术机密、伪造实验数据或违反科研伦理的学者进行追捕。

### **六大学部及其职能**

学会在日常决策中，通常受六大学部之间利益平衡的制约：

| 学部名称 | 核心研究领域与资产 | 主要职能与定位 |
| :--- | :--- | :--- |
| **基元象理与数学学部** | 研究高等数学、魔网场论与预测模型。 | 负责底层计算逻辑的开发，并通过算力验证服务与其他学部协作。 |
| **工程与系统象理学部** | 负责能量提取、魔工机械与大规模生产线设计。 | 学会内主要的经费来源部门，与帕特里欧重工等外部工业资本有较多合作。 |
| **地学与资源象理学部** | 负责地质、资源勘探与地下工事设计。 | 负责管理远征队与边疆哨所，负责野外测绘与地理数据收集。 |
| **生命与奥术生态学部** | 研究生物与灾厄生态，维护种质与标本库。 | 开展粮食安全与净化技术研究，是学会在社会公益领域的主要职能代表。 |
| **前沿项目与综合应用学部** | 研发魔算机芯片、加密网络、控制算法，并统筹重大跨学科工程。 | 规模较大且职能多元，前沿项目与信息算思是其核心研发内容。 |
| **社会象理与历史/统计学部** | 研究阶级结构、技术扩散与战争经济学。 | 负责政策评估，部分学者长期关注底层劳工调查。 |

### **权力结构与内部博弈**

#### **象理中央理事会**

学会的最高决策在程序上由象理中央理事会主导，日常决策常表现为三大核心政治派系在各学部权能与筹码交织下的动态拉锯：

*   **纯学术派（以理论自治与中立保护为核心）**：
    主张学术研究去政治化与科学中立。该派系以拥有松散理论网络的**基元象理与数学学部**、掌控基础地质与魔流数据的**地学与资源象理学部**，以及维持独立实验室自治的**生命与奥术生态学部**为核心。其政治筹码在于对基础研究和核心技术标准的解释权。例如，数学学部可通过拖延“算力验证服务”使不符合安全规范的魔工设计案搁浅；生命学部的独立研究所常以信息滞后或安全危害为由延迟上报实验数据，以规避外部审查；地学学部则有权以魔网失稳为由将特定矿产区域判定为不可开采区。
*   **工业与资本派（以商业变现与集权垄断为核心）**：
    倾向于科研成果的商业化与军事化应用。该派系以掌握巨额科研预算与实业车间的**工程与系统象理学部**，以及分配前沿先导项目合同的**前沿项目与综合应用学部**为骨干，与帕特里欧重工等世俗资本合作深厚。其政治筹码在于经费预算的绝对分配权与行政审查权。该派系常通过定向削减纯理论或社会学研究的经费预算，或利用信息技术项目的安全审计权，强迫其他学部向世俗军事和商业订单低头。
*   **社会与工会派（以技术开源与劳工保护为核心）**：
    主张技术去垄断化、科研公共服务化。该派系以拥有广泛基层调查网络的**社会象理与历史/统计学部**、基层地质外勤调查员以及底层的学徒、技工为基础。其核心筹码在于对基层社会的舆论动员能力与数据话语权。社会历史/统计学部可随时在《象理年鉴》或独立报告中公开真实的工厂工伤率与军事战损册，从而引发社会工潮或舆论风暴；底层技工则在“圆规与齿轮”等秘密工会的协调下，通过技术性干预手段（如调整高危设备安全参数引发自动锁闭）对抗过度剥削。

#### **象理纠察队与科学诚信与伦理委员会**

学会内部的安全司法权处于一种二元制衡结构中：
*   **象理纠察队**：作为执行跨国司法与安保任务的武装力量，其预算与行政指令受经费与规划局（工业派主导）监管。在实际运作中，纠察队常被用作工业派推进技术专利垄断、收捕技术泄密者以及查封“圆规与齿轮”秘密工会的行政铁拳。
*   **科学诚信与伦理委员会**：作为纠察队执行逮捕、搜查及封存实验数据的前置审批机构，该委员会席位主要由纯学术派的资深学者占据。伦理委员会利用其独立审核签字权，对纠察队的越界搜捕进行抵制，实质上充当了争议学者与民生科研成果的保护伞。

#### **横向机关与外部游说**

世俗政权与财阀等外部力量虽然没有理事会的直接投票权，但常通过定向科研捐赠与联合工程投资，向经费与规划局及国际与边疆事务局进行资金倾斜，以此游说各学部决策者。这使得学会的部分横向办事机关成为利益博弈的游说窗口。

### **边疆台站的分化与冲突**

<div class="faccordion">

#### **边疆台站的本地化倾向**
远离总部的边疆台站受地理隔阂与资源分布的影响，在内部博弈中呈现出多态分化的生存逻辑：
*   **资源本地化（常态避难所与技术中介）**：在远离权力中心的偏远台站（如沙海、雾幻海边缘），站长常利用地理隔离，将其转变为学术自治与社会派的避难所，收容被纠察队追捕的“圆规与齿轮”成员或生命学部的改革派学者。在中央预算因博弈断供时，大部分台站会提供弹道计算服务和地质报告以维持生存。
*   **资源富集区拉锯（开采与抵制）**：在稀有矿产与镜玉富集的台站，中央“工业派”试图通过大笔定向投资，将台站收归为大型矿业集团的开采前哨，而基层科学家则极力联合当地社区抵制，试图维持其民生防灾与中立监测职能。

#### **圆规与齿轮秘密工会**
圆规与齿轮是活跃于学徒、青年调查员以及工厂工人之间的非正式学术工会。
*   **主要诉求**：主张技术去垄断化、开源化以及科学研究的公共利益化。
*   **维权方式**：在应对高负荷劳动或利益分配不均时，工会成员曾采取技术性干预手段，例如调整高危设备安全参数。此外，工会秘密整理净水、无烟炉灶等民生技术的手册向民间分发，并协助部分被追捕的学者转移。

#### **奥术植物学的内部争议**
奥术植物学方向是生命与奥术生态学部内部争议较多的领域：
*   **镜玉作物改良**：受工业部门与外部资本支持的保守派学者，研发了具有高产量但无法留种特征的杂交镜玉作物。该作物的种植对特定魔力催化剂有高度依赖，这一技术路线曾被批评为过度捆绑自耕农的经济主权。
*   **抗灾作物研发**：与社会派学者合作的改革派人员，致力于培育可自由留种、适应低魔力环境的抗灾作物，并在废弃矿区开展土壤毒素过滤实验。
*   **非常规交易**：为弥补研究资金不足，部分边疆台站的植物研究人员涉嫌私自采摘特定区域的异相魔药，避开内行司的税收审核，向民间市场或雇佣兵组织输送，甚至用于提炼炼金药剂。
*   **高危项目指控**：有记录表明，极少数实验室在外部势力资助下，曾进行异相植被的武器化研究，其中包括诱发植被吸蚀魔网能力的孢子项目。这类高危项目在学会内部常面临严厉的伦理审查。

### **历史性灾难与路线战役**

#### **帝国崩溃期的技术清洗（215纪年）**
在帝国崩溃期间，学会内部曾因是否向世俗势力移交前代终极奥术武器的研究成果而产生严重分歧。坚守中立的学者在安全部队的协助下，对主张移交技术的派系进行了技术性清除，并将相关资料封存。此事件确立了学会禁止研发攻击性战略武器的基本原则，但亦导致部分技术人员流失。

#### **红泉协定大罢工（218纪年素秋）**
在中部镜玉加工厂发生多起辐射伤害事件后，由于管理层对数据的处理引发不满，学徒与工人发起了波及多处台站的联合停工。大罢工导致部分魔网稳定工作停滞，交通网络受到影响。理事会最终达成妥协，签署《红泉协定》，承认了学徒代表在管理中的话语权，并在理事会中增设了工人代表席位。

#### **三色石锚坍塌事故（219纪年）**
前沿项目学部在与东赐地方合作进行跨国魔网稳定锚并网实验时，因计算模型未能充分预测地下魔力回涌，导致石锚过载并引发局部魔网剧烈坍塌。事故造成实验区域环境恶化并出现大范围侵蚀现象。此事件直接促成了科学诚信与伦理委员会的权限扩大，确立了高危实验的一票否决制度。

#### **正在进行的源魔网论战**
学会内部目前在魔网利用路线上存在两种主要学术观点：
*   **稳定网派**：主张通过铺设稳定锚，将野性魔网规整为高安全性但低能效的标准魔网，保障大工业与民众的日常安全。
*   **野魔能派**：认为过度规整会降低魔力上限并抹杀技术突破的可能性，主张直接利用原始高能魔网。目前，双方在野外实验场地的申请与技术监督上存在较多冲突。
