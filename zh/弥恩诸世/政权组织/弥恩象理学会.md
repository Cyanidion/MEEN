# **弥恩象理学会**

弥恩象理学会是当代弥恩世界规模较大的跨国科研组织之一，由旧弥恩帝国晚期的帝国象理学会与帝都大学演变而来。旧帝国解体后，原有的国家研究体系发生分化，部分研究人员与术士为了自保与维持研究，以象理学会为名重组了跨地域的协作网络，将零散的实验室、学部与魔工工厂重新串联。

学会的研究范畴包括高等数学与基元场论等抽象理论，以及魔网流动、灾厄生态与社会系统中的运行规律，主导用统一的象理框架解释魔法、工业与社会变迁。该组织强调实验数据共享与工程可复现性，主导知识的公共开发与技术开源。

目前，象理学会在中部监护政府、南部帝国、云山诸领与北漠城邦都设有附属研究站与象理联络处。其定位在常规世俗政权之外具有较高的独立性，但在运行中，不同政权与资本集团常被认为在学会内部扶植各自的路线。

学会内部以六大学部为主干：
*   **基元象理与数学学部**：研究高等数学、魔网场论与预测模型，负责底层计算逻辑，提供算力验证服务。
*   **地学与资源象理学部**：负责地质、资源勘探与地下工事设计，管理远征队与边疆哨所，收集地理数据。
*   **生命与奥术生态学部**：研究生物与灾厄生态，维护标本库，开展粮食安全与净化技术等社会公益研究。
*   **工程与系统象理学部**：负责魔工机械、自动化控制与大规模生产线设计，是学会内主要的经费来源部门。
*   **社会象理与历史/统计学部**：研究阶级结构、技术扩散与社会统计，评估政策并长期关注底层劳工调查。
*   **前沿项目与综合应用学部**：统筹跨学科重大工程（如边疆勘探、魔网稳定锚工程）及前沿先导项目。

## **义务**

1. 根据规定，在象理学会名义下进行的调查、实验与工程项目，其原始数据、样品与技术说明需在一定时间内递交学会档案库*
2. 利用学会资源完成的研究成果，通常需在学会内部刊物上优先发表*
3. 成员需遵守伦理守则，在未评估风险的情况下不得释放未知灾厄样本或高危构造体，同时禁止以伪造数据谋取利益*
4. 当所在地区发生与魔网失稳、镜玉污染或重大灾厄相关的事故时，成员通常有义务向最近的象理联络处报告，并在能力范围内提供协助

**基元象理与数学学部**
根据学部规定，每年需投入约10天参与学部组织的学术复算或评审。在冒险中收集特定魔网参数，经交涉检定（DC25），可计入该义务时间。

**地学与资源象理学部**
每年需至少参加一次由学部组织的野外联合测量，负责记录地质与魔网信息。无正当理由缺席可能导致其在学部内的声望受限。

**生命与奥术生态学部**
负责维护所属实验站的活体与标本库，定期提交生态监测与样本繁殖报告。所有野外样本需通过安全隔离评估。

**工程与系统象理学部**
对所有以学会标准图纸生产的设备进行年度安全复核，并向学会提交改进建议。

**社会象理与历史/统计学部**
每年须提供至少一份关于本地区阶级结构、技术扩散与舆论动向的书面报告，且成员必须声明自己与地方政权、资本或革命组织的利益关联。

**前沿项目与综合应用学部**
在战时或重大工程期内，接受学会指派前往前线指挥部或重点工程局担任象理顾问。

> [.is-warning]
> *若成员长期不履行上述标记有星号（*）的义务，除非角色能够提供合情合理的解释（例如被长期俘虏、重伤或被迫隐匿），否则将被暂停特权甚至除名。

## **特权**

**通用特权**
* 凭成员身份，在各地象理合作商行 and 学会挂牌的工厂内，购买标准仪器、基础魔工品与科研用品时享受九折优惠。

**基元象理与数学学部**
* 在处理理论魔网模型时，每日一次，可在知识（奥秘）或知识（数学或工程）检定中获得 +4 见识加值。

**地学与资源象理学部**
* 学会野外站与附属矿山对成员开放，成员可以免费借用标准测量设备；在荒野或地下进行生存与导航检定时获得 +2 见识加值。

**生命与奥术生态学部**
* 有权从学会种质库中申请有限量的实验材料与低危奥术植物种子；在与自然或灾厄生物互动的交涉、荒野认同检定时获得 +2 加值。

**工程与系统象理学部**
* 制造复杂魔工装置的工艺检定视为在良好工具与完善资料条件下进行，且部分材料费用可以申请报销。

**社会象理与历史/统计学部**
* 在多数文明国家被视为中立技术人员，在与官僚、工会或商会领袖进行的交涉检定时获得 +2 加值。

**前沿项目与综合应用学部**
* 在执行学会重大项目任务时，可依授权调用后勤保障或随行人员，并在涉及跨学科协作的检定上允许一次重掷。

## **加入部门条件**

| 部门 | 条件 |
| :--- | :--- |
| **基元象理与数学学部** | 知识（奥秘）8级，文件解读5级，或具备法术叠变与法术加权等专长 |
| **地学与资源象理学部** | 生存5级，知识（地质或工程）8级，或能释放2级以上地变系法术 |
| **生命与奥术生态学部** | 知识（大自然）8级，医疗5级，或具备植物与野性相关背景 |
| **工程与系统象理学部** | 知识（高科或工程）8级，工艺（制造机械或导魔）5级 |
| **社会象理与历史/统计学部** | 知识（历史或经济）5级，交涉或收集信息8级 |
| **前沿项目与综合应用学部** | 任意三门知识8级，且获得另外两个学部常务理事的推荐信 |

> **特殊**：该专长不可被重训，不可以通常方式获取，不可因任何原因失效。

## **世界地位**

在旧帝国解体后，各地区武装势力普遍向学会寻求魔网稳定与弹道计算等技术协助。不过，纠察队享有的跨境司法特权在一定程度上也引发了地方统治者的戒备。在能量工业上，学会与帕特里欧重工等大资本合作较多，但底层技术人员建立的非正式维权网络，使学会的运作在外部行会眼中带有一定的不确定性。此外，学会与弥恩地理学会、天地稽古协会等机构保持着学术往来，并在部分重叠性项目上存在学术竞争。

象理学会的意识形态为**进步，集权，科学，干预**。
