# 不屈斗志

**学派**：防护系

**等级**：自我 4，牧师 4

**法术成分**：言语，姿势

**施法时间**：迅捷动作

**射程**：接触

**目标**：一个生物

**持续时间**：1 分钟/等级

**豁免检定**：意志通过则无效

**法术抗力**：可

你呼唤内心深处不屈不挠的战斗意志，将精神力量提升至超越肉体痛苦的境界。在法术持续时间内，你免疫晕眩、恶心、战栗状态。当你的生命值因伤害而降至 0 以下但并未死亡时，你不会陷入昏迷状态，你可以继续正常行动，但每轮结束时，你会因伤势而受到 1 点伤害，此伤害无法以任何方式减免或避免。在此状态下，如果你的生命值因受到治疗而恢复到 0 或 0 以上，此效果中止，直到你的生命值再次降至 0 以下时重新生效。此效果不能让你免于因巨创（通常是 -10 HP 或体质流失）而立即死亡。
