# 空山

*乐器（尺八），传说（需同调）*

一支由中空古竹制成的尺八，内壁刻满早已失传的音律符号，传说由南境的乐器制造大师[无心](/弥恩诸世/人物汇总/无心)打造。进行演奏时，其悲怆的音乐会使周围魔网产生律动，带来独特的听觉体验。

在你携带此物体时，你获得以下特性：

- **乐场**。你的法术攻击检定与法术豁免DC获得+2加值。
- **共鸣**。你可以使用一个动作演奏尺八，并进入名为“共鸣”的专注状态。你可以正常施法，并使用乐声替代施法成分中的言语（V）。每次长休你能使用的次数等于你的熟练加值，每次短休恢复一次使用次数。

在**共鸣**状态中，你可以选择对10尺范围内能够听到乐声的所有盟友造成以下任一效果：

- **胜绝**。在你的下一个回合前，所有攻击检定获得等于诗人激励骰的加值。
- **双调**。下一次攻击命中时，将伤害翻倍。
- **鸾镜**。恢复2*诗人激励骰+你的魅力调整值点生命值。
- **凫钟**。在你的下一个回合前，速度翻倍。

同时，你可以令10尺范围内能够听到乐声的所有敌人进行一次感知豁免检定，并对失败者造成以下任一效果：

- **断金**。下一次攻击具有劣势。
- **下无**。在你的下一个回合前，获得对任意一种伤害的易伤。
- **盘涉**。在你的下一个回合前，移动速度减半。
- **壱越**。调整其行动顺序。
