# 特殊能力

基于魔网本质理论，弥恩人在法术解密运动中，对各种特殊现象与能力进行了研究与诠释。以下特殊能力分类用于确定能力是否受到驱散、法术抗力、反魔场、魔网稳定和其他规则影响。

若某项能力没有明确类型，地下城主应根据其来源、表现形式和规则文本指定类型。无法归入下列类型的能力，默认为其他能力 Ot。

## 特殊能力类型

<h3 id="特异能力">特异能力（Extraordinary Abilities，Ex）</h3>

特异能力不含魔法。此类能力通常来自训练、生理结构、战斗技巧、非魔法器官或特殊习性。能干扰或中断魔法的方式和区域不会对特异能力产生作用。

特异能力不受驱散、法术抗力、反魔场或魔网稳定影响，且通常不因使用而引发借机攻击。

<h3 id="类法术能力">类法术能力（Spell-Like Abilities，Sp）</h3>

类法术能力是类似施法的魔法能力。它们不一定需要言语、姿势或材料成分，但其本质仍然是通过魔网调用法术型效果。

类法术能力在魔法受到抑制或无效的区域内无法运作，会受到解除魔法、法术抗力、反魔场和魔网稳定影响。使用类法术能力通常会引发借机攻击，除非能力说明另有规定。

<h3 id="超自然能力">超自然能力（Supernatural Abilities，Su）</h3>

超自然能力带有魔法性质，但不完全等同于类法术能力。它们通常来自生物本质、职业训练、血脉、神秘器官、魔网共鸣或特殊存在状态。

超自然能力在魔法受到抑制或无效的区域内无法运作，也会受到魔网稳定影响。与类法术能力不同，超自然能力不受普通解除魔法影响，也不受法术抗力影响，且通常不因使用而引发借机攻击。

<h3 id="灾厄异能">灾厄异能（Apocalyptic Abilities，Ap）</h3>

灾厄异能是灾厄类生物或灾厄化存在表现出的异常能力。此类能力并非通过有序魔网调用力量，而是来自灾厄污染、灾化结构或世界秩序崩坏后的异常机制。

灾厄异能在魔法受到抑制或无效的区域内通常仍能运作，不受普通解除魔法、法术抗力或反魔场影响。相反，在高度稳定或经特殊净化的魔网环境下，灾厄异能可能受到压制。具体压制方式由能力条目、地区规则或地下城主裁定。

<h3 id="赝作魔网异能">赝作魔网异能（Artificial-Weave Abilities，Ar）</h3>

赝作魔网异能是里界住民或赝作魔网相关存在表现出的特殊能力。此类能力依赖赝作魔网，而非弥恩通常意义上的有序魔网。

赝作魔网异能只有在赝作魔网条件下才能正常运作。普通解除魔法通常无法影响此类能力，但解除魔法（赝作）或其他专门针对赝作魔网的手段可以对其生效。反魔场、魔网不稳定和稳定魔网压制是否影响此类能力，由能力来源和具体区域性质决定。

<h3 id="源魔网能力">源魔网能力（Original-Weave Abilities，Or）</h3>

源魔网能力源于始源魔网，直接触及魔网更深层的原始结构。此类能力不受普通驱散、法术抗力与魔网不稳定影响，也不会因一般有序魔网紊乱而失效。

源魔网能力仍然依赖魔网本身。在反魔场、魔网被完全抑制、隔绝或无效化的区域内，源魔网能力将无法作用。只有明确标注为 Or 的能力或法术，才适用本条规则；“依赖源魔网”并不自动等同于“属于源魔网能力”。

<h3 id="超魔网能力">超魔网能力（Over-Weave Abilities，Ov）</h3>

超魔网能力是一种特殊的魔法能力，能够绕过一般有序魔网环境的限制。此类能力在魔网受到抑制或无效的区域内通常仍能正常运作，也不受普通法术抗力影响。

超魔网能力仍然可以受到解除魔法或类似驱散手段影响。除非能力说明另有规定，使用超魔网能力不引发借机攻击。

<h3 id="其他能力">其他能力（Other Abilities，Ot）</h3>

其他能力泛指无法归入上述分类的特殊能力。一般未详细说明的其他能力都不受法术抗力、反魔场与驱散影响，且不因使用而引发借机攻击。

地下城主可以根据具体能力的来源，将其他能力重新指定为更合适的特殊能力类型。

# 魔网稳定

只有在稳定的魔网之下，秩序才能存在。魔网是法术、类法术能力、超自然能力和魔法物品的基础。离开魔网，法术与能力都将彻底失效，魔法物品与奇物也将逐渐失去效用。

表示魔网稳定程度的数值称为**魔网稳定值**（Weave Stability，简称 **WS**）。WS 的范围通常为 0 至 100。WS 100 表示魔网完全稳定；WS 0 表示魔网彻底紊乱、崩溃或无法被有序调用。

在弥恩的绝大多数土地上，魔网都是绝对稳定的。地区效应、灾厄污染、魔网战争遗留、特殊武器、魔网稳定锚、导魔事故或其他因素，可能导致某一区域的魔网发生波动，形成魔网不稳定区域。

## 环境魔网稳定与自体魔网稳定

**环境魔网稳定**指某一区域的 WS。它决定该区域内魔法现象、持续性环境法术、魔法物品和导魔结构能否稳定运作。

**自体魔网稳定**（Self Weave Stability，简称 **SWS**）指生物或特殊个体自身承载的魔网稳定程度。SWS 主要影响作用对象为自体的魔法、特殊能力与魔网依赖状态。

在绝大多数情况下，生物的 SWS 与外界 WS 相同，因此可以等同看待。但某些特殊状态、能力、药物、污染、魔网稳定锚或装备会导致 SWS 高于或低于外界 WS。

## 魔网稳定变化

在魔网稳定锚存在的区域内，魔网会自行稳定。除非地区规则另有规定，该区域 WS 每日固定上升 10，最高不超过 100。

若没有魔网稳定锚，区域魔网即使被暂时稳定，也会逐渐回落。除非效果说明另有规定，该区域 WS 每小时衰减 10，直到回到该区域的自然稳定值。

智慧生物可以作为魔网的载体。生物的 SWS 每小时会自然向外界 WS 靠近 10，直到与外界 WS 相同。若有特殊能力、装备或状态维持 SWS，则按该效果说明处理。

## 魔网依赖

对于弥恩生物而言，魔网稳定是一种可感知的环境效应。只有绝对稳定的魔网才是最适宜的环境；魔网越不稳定，对生物造成的影响越大。

这种影响与生物的**魔网依赖**有关。魔网依赖是衡量一个物种、模板或存在状态对魔网稳定敏感程度的数值。魔网依赖越高，魔网稳定变化造成的影响越严重。

# 魔网混乱法术效应

在魔网不稳定区域中，施法会变得更加困难。随着魔网混乱程度加深，法术本身也有被扭曲、反噬或异常强化的可能。

当相关稳定值低于 100 时，使用任何法术或法术型效果都要进行一次**魔网混乱检定**。

## 魔网混乱检定

进行魔网混乱检定时，投掷 d100，并与相关稳定值比较。

<table class="mien-equipment-table">
  <thead>
    <tr>
      <th>d100 结果</th>
      <th>结算结果</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>结果 ≤ 相关稳定值</td>
      <td>法术正常释放。</td>
    </tr>
    <tr>
      <td>结果 > 相关稳定值</td>
      <td>法术释放失败。</td>
    </tr>
    <tr>
      <td>结果 > 相关稳定值，且结果 ≥ 90 + 相关稳定值 / 10</td>
      <td>法术释放失败，但不只是简单失败；改为触发一次魔网混乱法术效应。</td>
    </tr>
  </tbody>
</table>

`90 + 相关稳定值 / 10` 不需要额外取整。由于 d100 结果本身为整数，直接按通常数学比较即可。

## 持续效果与稳定值下降

已处于作用中的法术效果，也会因魔网稳定度变化而概率失效。

每当受术者 SWS 或法术所在区域 WS 因非自然衰减而降低时，需对当前相关的每一个持续性法术效果进行一次魔网混乱检定。若检定失败，该法术效果解除。

自然衰减通常指 SWS 每小时向环境 WS 靠近，或无稳定锚区域内临时提高的 WS 正常回落。非自然衰减通常指灾厄、魔网武器、封魔效果、乱魔事件、魔网污染、特殊法术或类似效果导致的突然降低。

# 特殊能力压制与耗散

部分特殊能力依赖稳定魔网，如类法术能力和超自然能力。在魔网稳定下降的区域，这些能力会受到压制或失去效用。在魔网稳定度极低的区域，部分特殊能力可能会耗散而永远失去。

## 锚定强度与最低稳定需求

为判定特殊能力能否维持，使用以下两个数值：

**锚定强度**（Anchoring Strength，简称 **AS**）：表示该能力与魔网或其来源结构之间的稳固程度。  
**最低稳定需求**（Minimum Stability，简称 **MS**）：表示该能力正常维持所需的最低稳定条件。

生物特殊能力的 AS 按下式计算：

**AS = MIN（生物 ECL × 5，90）**

其最低稳定需求为：

**MS = 100 - AS**

ECL 越高，AS 越高，MS 越低，说明该生物的特殊能力越能在低稳定环境下维持。

若生物没有 ECL，地下城主可以使用 HD 代替 ECL。若能力来自职业、模板、血脉或特殊存在状态，地下城主也可以按照该能力对应的有效等级计算 AS。

## 特殊能力压制

特殊能力能否使用取决于该能力类型、生物魔网依赖程度，以及该生物当前 SWS。

除非能力类型或具体条目另有说明，类法术能力和超自然能力适用下列规则。

## 特殊能力耗散检定

当特殊能力需要进行耗散检定时，每日针对每一项持续性特殊能力投掷 d100。

若结果 ≤ MS - SWS，则该特殊能力永久失去。

若该特殊能力由职业提供，则该能力可因升级、重新训练、特殊仪式或地下城主允许的恢复手段再次获得。若该特殊能力来自模板、血脉、种族本质或独特存在状态，则通常不能通过普通方式恢复。

# 魔法物品压制与耗散

魔法物品的主动效果与持续性效果通常依赖稳定魔网。启用魔法物品时，若该效果为法术效果，则形同施法，需要进行魔网混乱检定。若该效果非法术效果，则使用本节规则。

## 魔法物品的锚定强度

魔法物品的 AS 按下式计算：

**AS = MIN（魔法物品 CL × 5，90）**

其最低稳定需求为：

**MS = 100 - AS**

拥有导魔特性的物品，其 CL 视为 5。若该物品拥有其他附魔或更高 CL，则取较高者。

拥有自稳特性的魔法物品不会在 WS > 0 的环境中失效，除非物品说明另有规定。

若魔法物品存在多个独立魔法效用，地下城主可以决定每项效用分别检定，或将其视作一个整体进行检定。

# 规则示例

<blockquote class="custom-blockquote-default">
一名 5 级法师位于 WS 60 的区域，且自身 SWS 也为 60。他施放火球术，目标区域同样为 WS 60。<br><br>本次施法相关稳定值为 60。法师投掷 d100，结果为 42，结果 ≤ 60，火球术正常释放。若结果为 72，则法术失败。若结果为 97，则结果大于 60，且结果 ≥ 90 + 60 / 10，即 96，因此触发魔网混乱法术效应。
</blockquote>

<blockquote class="custom-blockquote-default">
一名角色通过魔网稳定锚获得 SWS 100，但他所在区域 WS 为 40。他对自己释放防护法术，相关稳定值为 SWS 100，法术正常释放。<br><br>若他向远处 WS 40 区域释放区域攻击法术，则该效果同时涉及使用者 SWS 与目标区域 WS，取较低值 40，需要以 40 作为相关稳定值进行魔网混乱检定。
</blockquote>

<blockquote class="custom-blockquote-default">
一件 CL 5 的导魔物品位于 WS 40 的区域。该物品 AS = MIN（5 × 5，90）= 25，MS = 75。<br><br>由于 WS 40 ≤ MS，但 WS 40 > MS / 2，即 37.5，因此该物品受到压制，但尚不需要进行耗散检定。若区域 WS 降至 30，则 WS 30 ≤ MS / 2，该物品每日进行一次耗散检定。耗散概率为 MS - WS = 75 - 30 = 45。每日投掷 d100，结果 ≤ 45 时，该物品魔法效用永久流失。
</blockquote>

## 简化方式三：自主裁定

若地下城主希望将魔网不稳定作为叙事压力，而非每轮结算内容，可以直接由地下城主决定何时触发魔网混乱。

地下城主可以在以下情境触发魔网混乱法术效应：

- 角色在乱魔区内连续施法，明显依赖魔法解决场景。
- 角色释放高环法术、范围法术、传送类法术、召唤类法术或其他高风险魔法。
- 角色在灾厄污染或稳定锚失效后继续施法。
- 角色施法出现大失败、专注失败、法术穿透失败或其他明显失控结果。
- 当前场景需要表现乱魔区的危险性与不可预测性。

采用此方式时，地下城主应在场景开始前提示玩家：“此区域魔网不稳定，过度施法可能引发混乱。”除非场景本身具有隐秘性质，不应在完全无提示的情况下以魔网混乱惩罚玩家。
