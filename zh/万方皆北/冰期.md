1787年的冰期出乎所有人的意料。

学界一致认可冰期的直接成因是阳光减少。来自索拉里斯的磅礴能量透过天幕——我们称其为[[#知界]]，温暖万物，孕育众生。然而知界的某些改变导致更多阳光被折射回燃素海。这微小的变化带来了灾难般的寒冷气候。与此同时，埃维陇两极的魔网失控了。失控的魔网让灰域蔓延，并在阿尔比昂东部和大陆南极分别唤起一场宏伟的、前所未有的风暴。

二者的共同作用使得埃维珑迅速来到冰期。随着这些风暴向内陆挺近，大陆南北地区陷入了长达数十纪的阴天。作物不生，植被枯萎，逃难的人群在这一时期向低纬度地区迁移。厚重的云层遮天蔽日，局部气候的巨变深刻影响了全球气候，而在某种微妙的平衡和魔法风暴的影响下，直到如今，冰期仍未结束。目前，埃维珑的高纬度地区经年累月都有风暴肆虐，雪代替了雨成为主要的水循环。

*由于恶劣的气候和医疗物资的匮乏，缺乏准备的野外行动可能受到严重的冻伤。严寒下，人性也流失更快。*

同样，在外出冒险时，角色应当考虑极端气候的影响。风暴同样会对阳光产生影响，突然的恶劣天气不仅带来低温，还有能见度的缺失。

## 应对寒冷
生命活动产生热量，而生命活动也需要一定温度才能持续。通常来说，在环境造成的热能损失可以被冒险者自身的产生的热量所弥补时，冒险者可以维持自己的体温。故而，只要装备足够保暖的服装，无论怎样的冒险者都可以长久在恶劣环境中生存。然而足够应对所有情况的御寒方案是不存在的，冒险者应对寒冷的能力由装备决定，受自身体质属性的影响。

规则给各类常见装备及其变种赋予了恒温等级，对于世设没有提到的各类衣物，DM可以参考这些后给予一个合理的数值。将这个数值加上人物的体质调整值，加上如火炉、暖水袋等热源提供的环境调整值，再加上避风处带来的额外增益，最终减去自然环境带来的环境修正值，假如这一数值为负，就得到人物在当前环境下的**恒温调整值**。如果数值为正，**调整值就是0**，意味着玩家在当前环境不受影响。玩家已经承担了额外的盔甲减值、购买御寒物资的额外资金和计算一大堆额外数值的心理负担。既然他们已经叠起一大堆加值，就对他们好一点吧。

衣物的恒温等级不像盔甲，可以叠加，最多同时享受两件衣物的加值。不过相应的，你需要承受所有衣物的防具减值之和。

推荐玩家预先计算好数值中相对固定的部分（体质+装备调整值），记在人物卡中方便查找。

在应对风暴的强韧豁免检定中，角色需要将结果减去**恒温调整值的平方**。

### 力竭
*这段规则修改自DnD 5e*

某些特殊能力和环境危害（例如遭受饥饿，或是长时间暴露在极热或极冷的温度下），会导致名为力竭的特殊状态。力竭状态分为六个递增的等级。当特定效应造成力竭时，根据其具体描述的不同可能令生物陷入一级或更高等级的力竭中。

在寒冷情况下，当你开始失温，严寒就更易侵蚀。身体产生热量的效率随失温而降低，衣物破损、雪水堆积或是冻伤导致的血液流动不畅也对健康产生影响。简而言之，冻伤导致的理解拥有第三栏的额外效果：

| 力竭等级 | 效果            | 恒温登记影响  |
| :--: | :------------ | ------- |
| 1    | 属性检定具有-2修正    |         |
| 2    | 速度减半          | 恒温调整值-1 |
| 3    | 攻击和豁免检定具有-2修正 |         |
| 4    | 生命值上限减半       | 恒温调整值-1 |
| 5    | 速度降为0         | 恒温调整值-1 |
| 6    | 死亡            | 死亡      |

已力竭的生物如果受另一效应影响而再次遭受力竭，则将其当前力竭等级与该效应造成的力竭等级相叠加。再度遭受高等级力竭的生物承受其当前力竭等级及所有更低级力竭的效应。例如，受2级力竭影响的生物属性检定具有-2修正，且速度减半。

移除力竭的效应按其描述降低力竭等级，力竭等级小于1时，所有力竭产生的效果结束、生物在获得足够的饮食，并在旅行场景外或是足够温暖处完成一次至少八小时的休息后，其力竭等级即可降低1级。

你每承受2级力竭，就失去一点人性。恢复力竭不会恢复失去的人性。

### 阻碍移动
被厚厚积雪覆盖的地形能够有效降低陆行生物的速度。在雪上行走时，大多数生物并非每次都会全沉进去。有时一层坚硬的冰壳可以有效的防止踩在上面的人的脚沉进积雪。另外，在松软的雪层下几英尺甚至几英寸处残留有旧落雪层，这些旧落雪层很可能硬到足以防止旅行者沉进去了。  表内说明了松软的积雪对行进造成妨害的等级，其中，“小”体型代表了小及以下的体型，“大”体型代表了大及以上的体型。**注意1英尺为12英寸**。

| 积雪深度    | 小型  | 中型  | 大型  |
| ------- | :---: | :---: | :---: |
| 最多6英寸   | 小   | 无   | 无   |
| 7-12英寸  | 小   | 小   | 无   |
| 13-24英寸 | 大   | 小   | 小   |
| 25-36英寸 | 大   | 大   | 小   |
| 37-60英寸 | 完全  | 大   | 小   |
| 61+英寸   | 完全  | 完全  | 大   |
**无**：积雪对生物的速度没有显著影响。  
**小**：人物需要花费两格行动力才能在积雪上移动一格(速度减半)。翻滚DC+2。  
**大**：人物需要花费四格行动力才能在积雪上移动一格(速度变为1/4)。翻滚DC+8。  
**完全**：人物除非通过一次DC 5的力量或平衡检定(人物自行选择)否则无法移动。人物移动成为一个全回合动作。人物需要人物需要花费四格行动力才能在积雪上移动一格(速度变为1/4)。滚翻DC +20，且人物的AC失去敏捷调整值。

### 法术及其他
法术或其他来源提供的*寒冷伤害抗性*每5点提供1点恒温等级，此外还有更改后的*忍受环境*法术。下面列举出一些法术：

#### 抵抗能量伤害 Resist Energy
*防护系*

**等级**：牧师 2，德鲁伊 2，火 3，圣武士 2，巡林客 1，术士/法师 2
**法术成分**：言语，姿势，法器
**施法时间**：1个标准动作
**距离**：接触
**目标**：**接触的生物**
**持续时间**：10分钟/等级
**豁免检定**：**强韧通过则无效（无害）**
**法术抗力**：可（无害）

对以下五种能量类别之一的伤害，此防护法术给予一个生物一定的有限保护，能量类别由你选择**：强酸、寒冷、电击、火焰或音波。该目标生物得到对于选定能量形式的10点能量抗性，意味着每次它受到该类能量伤害（无论是来源于自然还是魔法）时，该生物的生命值受到的伤害降低10点。此能量抗性在第7级时增加为20点，第11级时达到最高，为30点。此法术也会保护目标生物的装备。

抵抗能量伤害_只吸收能量伤害。目标仍然会受到其他负面效果的影响。

**注意**：防护能量伤害*覆盖*（而非叠加）了抵抗能量伤害（__Resist Energy__）。如果一个人被_防护能量伤害_和_抵抗能量伤害_所同时防护，那么_防护能量伤害_会吸收伤害，直到它的能力消失。

#### 防护能量伤害 Protection From Energy
*防护系*

**等级**：牧师 3，德鲁伊3，机运领域 3，保护领域 3，巡林客2，术士/法师 3
**法术成分**：言语，姿势，法器
**施法时间**：1个标准动作
**距离**：接触
**目标**：**接触的生物**
**持续时间**：10分钟/等级或直到耗散
**豁免检定**：**强韧通过则无效**（无害）
**法术抗力**：可（无害）

*防护能量伤害*能提供暂时的对一种能量的免疫力，能量类型由你在施法时指定（强酸，寒冷，电击，火焰，或音波）。当这个法术吸收了每施法者等级12点的能量伤害时（在10级时达到最多120点），法术耗散。

如选择寒冷，防护能量伤害在应对寒潮时视作一个可用次数为施法者等级一半的消耗品，在10级达到最多5次，提供施法者等级一半的恒温等级加值。每消耗一次可用次数，它提供的恒温等级加值减少1，视作这个法术吸收了寒冷对你造成的损伤。

**注意**：防护能量伤害*覆盖*（而非叠加）了抵抗能量伤害（__Resist Energy__）。如果一个人被*防护能量伤害*和*抵抗能量伤害*所同时防护，那么*防护能量伤害*会吸收伤害，直到它的能力消失。

#### 忍受环境 Endure Elements
*防护系*

**等级**：牧师 1，德鲁伊 1，圣武士 1，巡林客1，术士/法师 1，太阳 1
**法术成分**：言语，姿势
**施法时间**：一个标准动作
**距离**：接触
**目标**：被接触的生物
**持续时间**：8小时
**豁免检定**：意志，通过则无效（无害）
**法术抗力**：可（无害）

被忍受环境所保护的受术者能够更好地对抗极端气候。他的恒温等级获得一个+2的加值。该生物的装备同样被保护。

### 风暴

风暴带来多方面的影响，首先，依赖视觉的人物可视范围减半，同时使侦察与搜索检定承受-4 的减值。在强风中，无论体型大小，范围内所有生物的远程攻击及「聆听」检定 都受到-2减值，烈风中则是-4减值。风势会吹灭蜡烛、火把和其他 没有遮罩的火焰。有遮罩的火焰（如提灯）会剧烈摇晃，且有50%机会熄灭。

此外，对于雪风暴而言，一整天的降雪能形成1d6英寸深的积雪。

> 强酸和音波攻击对大多数物品造成的伤害和生物 一样，伤害值等于伤害掷骰结果减去物品硬度。电击和火焰 对大多数物品只会造成一半伤害，伤害值等于伤害掷骰结果 除以2再减去物品硬度。寒冷对大多数物品只会造成四分之一伤害，伤害值等于伤害掷骰结果除以4再减去物品硬度。  远程武器伤害：远程武器（攻城器除外）对物品只会造成一半伤害，伤害值等于伤害掷骰结果除以2再减去物品硬度。

在大型风暴中，或许需要考虑它对物品的影响。此外，在埃维珑，存在因魔网紊乱引起的反常风暴……最好不要随意涉足其中。

不同种类的风暴有着不同的恒温等级修正值。玩家需要通过等同于其恒温检定修正值的强韧豁免检定，如前文所述，这一检定受到玩家**恒温等级调整值平方**的减值。

#### 雪风暴
**检定DC**：12～15
**介绍**：埃维珑大地最常见的气象灾害，寒风和低温。进行一次强韧豁免，豁免失败承受1-2级力竭，并受到2d6点寒冷伤害，豁免通过承受0~1级力竭，伤害减半。在雪风暴中，积雪深度或许会随行程而增加。

#### 暴风雨
**检定DC**：10～13
**介绍**：尽管雪风暴占据了绝大多数灾害频率，暴风雨时有发生。在没有足够雨具的场合，暴风雨无视棉衣与盔甲带来的恒温等级，而防雨衣物可以移除这一减益。进行一次强韧豁免，豁免失败承受1-2级力竭。

#### 魔网风暴
**检定DC**：18～20
**介绍**：快逃！
魔网风暴无视体质外的任何非魔法的恒温等级加值。魔法物品在其中有1/5的概率失效。魔网风暴的负面效应是多样的，包括心灵伤害、火焰伤害、寒冷伤害、多级力竭效应、负面状态及精神失常等。高强度的魔网风暴可能无视豁免，而无论强度，任何在豁免检定中失败的人都会失去部分人性。
